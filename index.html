jQuery FullBG

Plugin jQuery para imagens de fundo full size.

Imagens retiradas do site Wallbase.cc.

Como usar

<div class="full-bg" id="full-bg">
    <img src="img/bg-1.jpg" class="full-bg-img" />
    <img src="img/bg-2.jpg" class="full-bg-img full-bg-img-hidden" />
    <img src="img/bg-3.jpg" class="full-bg-img full-bg-img-hidden" />
</div>

<script>
    $(function() {
        $('.full-bg').fullBG();
    });
</script>

Opções

Opção Default Descrição
$img .full-bg-img Classe das imagens
$nav .full-bg-nav Classe da div de navegação
$navItem .full-bg-nav-item Classe do elemento da navegação
imageWidth 1500 Largura das imagens (em pixels)
imageHeight 750 Altura das imagens (em pixels)
transition true Se deseja que as imagens fiquem trocando automáticamente
delay 5000 Tempo para as imagens trocarem
speed 500 Velocidade da transição (fade)
imgAtual 1 Imagem que começar
nav true Se deseja utilizar a navegação entre as imagens
onWindowChange null Pode se passar uma função para ser executada no resize da janela.
